Your progress will auto-save to ironlog_progress.json inside a Workout Tracker folder in your Drive after every set. One-time setup takes ~10 minutes.
⚡ You only need to do this once. After setup, just open the file and it syncs automatically.
1
Go to console.cloud.google.com → Click Select a project → New Project → Name it IronLog → Create
2
In the left menu → APIs & Services → Library → Search Google Drive API → Click it → Enable
3
APIs & Services → OAuth consent screen → Choose External → Fill app name (IronLog) → Add your Gmail as test user → Save
4
Credentials → Create Credentials → OAuth 2.0 Client ID → Application type: Web application → Under Authorised JavaScript origins add your hosted URL (e.g. https://yourname.netlify.app) → Create
5
Copy the Client ID (looks like: 123456-abc...apps.googleusercontent.com) and paste below
